Tesseract OCR


          

            
            

              

                

                  This app now uses Tesseract.js locally for all scans, both online and offline. No external OCR service or API key is required.
                

                

                  Ensure node_modules/tesseract.js/dist/worker.min.js, tessdata/, and node_modules/tesseract.js-core are available when serving the app.
                

              

            

          


          


          

            
              
              Setup guide
            
            OCR engine: Tesseract.js local recognition

            No key required: works without external services

            Storage: contacts saved to localStorage as cs_contacts

            Request format: no network request — all OCR runs locally

            Console: press F12 → Console to see OCR worker logs and debug output

            Contacts: saved locally and editable before save

            Email: EmailJS config still stored in cs_ejs_service, cs_ejs_template, cs_ejs_pubkey
